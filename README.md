# 폴더 설명 및 파일설명
## c2j-pilot-parts
- c2j 앱이 저장되어있는 이미지
- 클로드코드, 딥에이전트 설치
- 용량 이슈로 분리해서 이미지를 나눠 놓음 합친 후 도커에 이미지 넣기

## cobol
- 테스트 코볼소스

## elipse-plugin-v6
- c2j-app이랑 연동되는 이클립스 플러그인

## 참고자료
- kesa 프레임워크 가이드

## ne4j.tar
- 그래프 db 구성용 이미지
- c2j-app와 연결해야 c2j가 쿼리 조회가능

## 세팅 예시
- 도커 설치 -> 이미지로 컨테이너 생성 -> 그래프 DB에 코볼 소스 및 가이드 적재
- c2j-app 컨테이너에서 클로드코드로 MAS 구성
